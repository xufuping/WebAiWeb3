### 💡 笔记哲学

- **三分钟原则**：每个知识点力求在三分钟内讲清楚核心概念
- **实践导向**：理论结合实践，每篇笔记都附带可运行的示例
- **持续迭代**：笔记不是一次性的，会随着认知升级不断更新
- **标签体系**：通过标签构建知识网络，形成立体的知识图谱
